CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • . card the period designates card as a class selector
  • class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class selector

Git

  • git status checks the branch we are currently working on
  • git checkout -b (name of branch) creates a new branch name
  • git add -A stages all commits to be pushed
  • git commit -m "label for commit"
  • git pull origin main makes sure we are working on most up to date
  • git push origin (name of branch)
  • Go to github repo to pull and merge commits.

JavaScript

  • Control Flow - the order in which a computer executes code in a script
  • A variable is a named container that allows us to store data in our code
  • if statement is decision making in code
  • Array - single variable that is used to hold a group of data
  • A for loop contains 3 important statements
  • 1st statement delcares starting point for our loop
  • 2nd statement is the condition
  • 3rd statement allow array to iterate over each item
  • A function is a set of instructions that tells a computer how to perform a certain task
